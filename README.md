## ¿Para qué sirve el archivo package.json?

El archivo `package.json` contiene la información del proyecto, como su nombre, versión, dependencias, scripts y configuración necesaria para que la aplicación funcione correctamente.

## ¿Qué diferencia notaron entre ejecutar un archivo directamente con Node.js y levantar el servidor de React?

Al ejecutar un archivo con Node.js (`node app.js`), el código se ejecuta directamente en la terminal y el resultado se muestra en la consola. En cambio, al levantar el servidor de React (`npm run dev`), se inicia un servidor de desarrollo que permite visualizar la aplicación en el navegador y actualiza los cambios automáticamente cada vez que se modifica el código.
